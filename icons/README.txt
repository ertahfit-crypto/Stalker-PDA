ВНИМАНИЕ: Нужно добавить иконки приложения

Для работы PWA необходимо добавить 2 файла PNG с изображением знака радиации:

1. icon-192.png (192x192 пикселей)
2. icon-512.png (512x512 пикселей)

Требования к иконкам:
- Квадратное изображение
- Знак радиации по центру
- Желтый знак на темном фоне (или наоборот)
- Без прозрачности (или с прозрачным фоном)
- Формат PNG

Можно использовать онлайн-генераторы иконок или создать в Photoshop/Figma.
