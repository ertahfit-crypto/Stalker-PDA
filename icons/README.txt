ВНИМАНИЕ: Нужно конвертировать SVG в PNG

Для работы PWA необходимы PNG файлы (SVG не поддерживается для PWA иконок).

Я создал SVG файлы со знаком радиации:
- icon-192.svg
- icon-512.svg

Нужно конвертировать их в PNG:

1. icon-192.png (192x192 пикселей)
2. icon-512.png (512x512 пикселей)

Как конвертировать:
- Открыть SVG в браузере
- Скриншот или использовать онлайн-конвертер
- Или использовать Figma/Photoshop/Inkscape

Или использовать онлайн-сервисы:
- https://cloudconvert.com/svg-to-png
- https://convertio.co/ru/svg-png/

После конвертации удалите .svg файлы и оставьте только .png
