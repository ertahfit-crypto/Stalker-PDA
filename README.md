# S.T.A.L.K.E.R. PDA Online

Полноценное веб-приложение в стиле игры S.T.A.L.K.E.R. с системой авторизации, чатом, картой зоны и торговлей.

## 🎮 Особенности

- **🔐 Авторизация**: Регистрация и вход сталкеров
- **💬 Онлайн-чат**: Реальное время через Firebase
- **🗺️ Карта Зоны**: Интерактивные локации (Кордон, Свалка, Агропром, Припять)
- **🎒 Торговля**: Рынок артефактов
- **👤 Профили**: Информация о сталкерах
- **🛠️ Админ-панель**: Управление системой

## 🎨 Дизайн

- Темная тема в стиле S.T.A.L.K.E.R.
- Неоново-зеленые элементы интерфейса
- Эффекты терминала PDA
- Анимации и переходы
- Адаптивный дизайн

## 🚀 Быстрый запуск

1. **Откройте `index.html` в браузере**

Приложение готово к использованию!

## 🔧 Firebase настройка (опционально)

Для работы с реальными данными:

1. Создайте проект в [Firebase Console](https://console.firebase.google.com/)
2. Включите Authentication (Email/Password)
3. Создайте Firestore Database
4. Замените конфигурацию в `app.js`:

```javascript
const firebaseConfig = {
    apiKey: "ВАШ_API_KEY",
    authDomain: "ВАШ_ПРОЕКТ.firebaseapp.com",
    projectId: "ВАШ_ПРОЕКТ",
    storageBucket: "ВАШ_ПРОЕКТ.firebasestorage.app",
    messagingSenderId: "ВАШ_SENDER_ID",
    appId: "ВАШ_APP_ID"
};
```
