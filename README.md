# СТАЛКЕР ЧАТ - Жёлто-чёрная индустриальная тема

Полноценный современный веб-сайт по вселенной S.T.A.L.K.E.R. с мрачным постапокалиптическим дизайном в жёлто-чёрной цветовой гамме.

## 🌟 Особенности

- **Индустриальный дизайн**: Грязный постапокалиптический стиль, ржавчина, радиация
- **Эффекты старого телевизора**: Scanlines, шум, vignette, лёгкое мерцание
- **Glitch-эффекты**: Жёлтые glitch-анимации в стиле старых мониторов
- **Аутентификация**: Регистрация и вход через Firebase Authentication
- **Реалтайм чат**: Терминальный стиль общения между сталкерами
- **Интерактивная карта**: 9 локаций из вселенной S.T.A.L.K.E.R.
- **Адаптивный дизайн**: Работает на всех устройствах
- **Жёлто-чёрная палитра**: Максимально атмосферный дизайн Зоны отчуждения

## 🚀 Быстрый запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка Firebase
1. Создайте новый проект в [Firebase Console](https://console.firebase.google.com/)
2. Включите Authentication (Email/Password)
3. Создайте Firestore Database
4. Скопируйте конфигурацию и замените в `app.js`:
   ```javascript
   const firebaseConfig = {
     apiKey: "YOUR_API_KEY",
     authDomain: "YOUR_PROJECT.firebaseapp.com",
     projectId: "YOUR_PROJECT",
     storageBucket: "YOUR_PROJECT.appspot.com",
     messagingSenderId: "YOUR_SENDER_ID",
     appId: "YOUR_APP_ID"
   };
   ```

### 3. Запуск сервера
```bash
npm start
```

Или для разработки:
```bash
npm run dev
```

### 4. Открытие сайта
Откройте в браузере: `http://localhost:3000`

## 🎮 Функциональность

### Главная страница
- Анимированный интерфейс PDA
- Статус Зоны в реальном времени
- Эффекты помех и glitch
- Терминальный вывод

### Карта локаций
- **Кордон** - Вход в Зону
- **Свалка** - Бандитская территория
- **Агропром** - Подземный комплекс
- **Тёмная долина** - Лагерь бандитов
- **Бар 100 Рентген** - Центр сталкеров
- **Янтарь** - Научная станция
- **Военные склады** - Охраняемая зона
- **Припять** - Мёртвый город
- **ЧАЭС** - Сердце Зоны

### Чат
- Реалтайм общение через Firebase Firestore
- Только для авторизованных пользователей
- Стиль терминала с временными метками

### Аутентификация
- Регистрация новых сталкеров
- Вход в систему
- Сохранение сессии

## 🎨 Дизайн и эффекты

### CRT эффекты
- Сканлайны (линии развёртки)
- Шумовой эффект
- Лёгкое мерцание
- Эффект старого монитора

### Glitch анимации
- Текстовые эффекты при наведении
- Случайные помехи
- RGB сдвиги
- Мерцающие элементы

### Цветовая схема
- **Основной**: `#00ff41` (зелёный PDA)
- **Вторичный**: `#ffb000` (янтарный)
- **Опасность**: `#ff3131` (красный)
- **Предупреждение**: `#ffaa00` (жёлтый)

1. **Firebase конфигурация**: Нужно заменить тестовые ключи на реальные
2. **Изображения локаций**: Используются placeholder'ы - можно заменить на реальные скриншоты
3. **Звуковые эффекты**: Не реализованы (можно добавить Web Audio API)
